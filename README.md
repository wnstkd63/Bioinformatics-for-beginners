## About

This repository documents my journey of learning bioinformatics from scratch as a biology undergraduate.

The goal is to provide beginner-friendly implementations and explanations so that students with limited programming experience can start bioinformatics more easily.

Each topic includes:

- Theory
- Python implementation
- Example input files
- Expected outputs
